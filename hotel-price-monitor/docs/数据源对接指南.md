# 酒店API数据源对接指南

## 概览

本项目采用**适配器模式**,支持多个酒店API数据源并行工作:

```
┌─────────────────────────────────────────┐
│        AdapterManager (管理器)          │
│   统一接口 + 价格比较 + 健康检查         │
└─────────────────────────────────────────┘
          │         │          │
    ┌─────┴──┐  ┌───┴────┐  ┌──┴─────┐
    │ Ctrip  │  │Amadeus │  │RapidAPI│
    │ 携程    │  │万豪凯悦│  │ Booking│
    └────────┘  └────────┘  └────────┘
```

## 1. Amadeus (推荐用于国际酒店)

### 注册步骤

1. 访问 https://developers.amadeus.com/register
2. 创建账号(免费)
3. 创建应用,获取:
   - **Client ID**
   - **Client Secret**
4. 选择环境:
   - **Test** (测试环境,免费)
   - **Production** (生产环境,需付费)

### 配置

在 `.env` 中添加:
```env
AMADEUS_ENABLED=true
AMADEUS_CLIENT_ID=your_client_id
AMADEUS_CLIENT_SECRET=your_client_secret
AMADEUS_PRODUCTION=false
```

### 覆盖的酒店集团

- **万豪** (Marriott International) - Chain Code: `MC`
- **凯悦** (Hyatt) - Chain Code: `HY`
- **洲际** (IHG) - Chain Code: `IC`
- **希尔顿** (Hilton) - Chain Code: `HL`
- **雅高** (Accor) - Chain Code: `AC`

### 价格限制

| 计划 | 免费额度 | 超出费用 |
|------|---------|---------|
| Free Tier | 2,500次/月 | - |
| Enterprise | 自定义 | 按量计费 |

### 示例代码

```javascript
const adapter = new AmadeusAdapter({
  clientId: 'xxx',
  clientSecret: 'xxx',
  production: false
});

// 查询万豪酒店价格
const result = await adapter.queryPrice({
  hotelId: 'ADPAR123',
  checkInDate: '2026-03-15',
  checkOutDate: '2026-03-16'
});

console.log(result.price, result.currency);
```

## 2. RapidAPI (快速接入多个OTA)

### 注册步骤

1. 访问 https://rapidapi.com/
2. 注册账号
3. 订阅 **Booking.com API**:
   - https://rapidapi.com/apidojo/api/booking-com
4. 获取 **RapidAPI Key**

### 配置

```env
RAPIDAPI_ENABLED=true
RAPIDAPI_API_KEY=your_rapidapi_key
```

### 定价

| 计划 | 请求次数/月 | 价格 |
|------|-----------|------|
| Basic | 500次 | $0 |
| Pro | 10,000次 | $10/月 |
| Ultra | 100,000次 | $50/月 |

### 优势

- ✅ 一次对接,接入 Booking.com / Priceline / Hotels.com
- ✅ 无需复杂认证流程
- ✅ 统一SDK,快速测试

### 示例

```javascript
const adapter = new RapidAPIAdapter({
  rapidApiKey: 'xxx'
});

const hotels = await adapter.searchHotels({
  keyword: '上海外滩',
  checkInDate: '2026-03-15',
  checkOutDate: '2026-03-16'
});
```

## 3. 携程 (国内酒店)

### 申请流程

⚠️ **需要企业资质**

1. 访问 https://open.ctrip.com
2. 企业认证
3. 提交商务合作申请
4. 等待审核(可能需要数周)

### 替代方案

如果无法获取官方API:
- 使用 RapidAPI 的携程数据(如有)
- 合规爬虫(需遵守 robots.txt)
- 使用飞猪替代

## 使用适配器管理器

### 初始化

```javascript
const AdapterManager = require('./src/services/AdapterManager');

const manager = new AdapterManager({
  ctrip: {
    enabled: true,
    apiKey: 'xxx',
    apiSecret: 'xxx'
  },
  amadeus: {
    enabled: true,
    clientId: 'xxx',
    clientSecret: 'xxx',
    production: false
  },
  rapidapi: {
    enabled: true,
    apiKey: 'xxx'
  }
});
```

### 查询单个数据源

```javascript
// 只查询携程
const result = await manager.queryPrice('ctrip', {
  hotelId: '12345',
  checkInDate: '2026-03-15',
  checkOutDate: '2026-03-16'
});
```

### 查询所有数据源(价格比较)

```javascript
// 自动比较所有数据源,返回最低价
const result = await manager.queryLowestPrice({
  hotelId: '12345',
  checkInDate: '2026-03-15',
  checkOutDate: '2026-03-16'
});

console.log('最低价:', result.lowestPrice.price, result.lowestPrice.provider);
console.log('所有价格:', result.allPrices);
```

### 健康检查

```javascript
const health = await manager.healthCheck();
console.log(health);

// 输出:
// {
//   ctrip: { status: 'ok' },
//   amadeus: { status: 'ok' },
//   rapidapi: { status: 'error', error: 'API key invalid' }
// }
```

## 扩展新数据源

### 步骤

1. 创建适配器文件: `src/adapters/YourAdapter.js`
2. 继承 `BaseHotelAdapter`
3. 实现必需方法:
   - `searchHotels()`
   - `queryPrice()`
   - `getHotelDetails()`

### 模板

```javascript
const BaseHotelAdapter = require('./BaseAdapter');

class FliggyAdapter extends BaseHotelAdapter {
  constructor(config) {
    super(config);
    this.name = 'Fliggy';
    this.apiKey = config.apiKey;
  }

  async queryPrice({ hotelId, checkInDate, checkOutDate }) {
    // 实现飞猪API调用逻辑
    const rawData = await this.callFliggyAPI(hotelId, checkInDate);
    
    // 标准化响应
    return this.normalizePriceResponse({
      hotelId,
      price: rawData.price,
      currency: 'CNY',
      available: true,
      checkInDate,
      checkOutDate
    });
  }

  // 实现其他方法...
}

module.exports = FliggyAdapter;
```

4. 在 `AdapterManager` 中注册:

```javascript
const FliggyAdapter = require('../adapters/FliggyAdapter');

initializeAdapters() {
  // ...其他适配器
  
  if (this.config.fliggy?.enabled) {
    this.adapters.fliggy = new FliggyAdapter({
      apiKey: this.config.fliggy.apiKey
    });
  }
}
```
