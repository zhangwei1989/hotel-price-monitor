**核心功能：**
1. **任务列表页面**
   - 显示所有监控任务
   - 支持筛选（城市、日期、状态、价格区间）
   - 支持搜索（酒店名称、房型）
   - 支持排序（入住日期、当前价格、创建时间）
   - 分页显示

2. **任务详情页面**
   - 任务基本信息
   - 价格历史记录（表格 + 图表）
   - 价格趋势图（折线图）
   - 操作按钮（暂停/恢复、删除、手动检查）

3. **仪表盘页面**
   - 任务总数统计
   - 成功率统计
   - 降价捕获数
   - 近7天价格趋势

4. **实时更新**
   - WebSocket 推送价格变化
   - 降价提醒弹窗

## 🔍 筛选功能设计

### 筛选条件

```typescript
interface TaskFilter {
  // 基础筛选
  city?: string;              // 城市（长沙、北京、上海...）
  status?: TaskStatus;        // 状态（活跃、暂停、已过期、低于目标）
  hotelName?: string;         // 酒店名称（模糊搜索）
  roomType?: string;          // 房型（模糊搜索）
  
  // 日期筛选
  checkInFrom?: Date;         // 入住日期起始
  checkInTo?: Date;           // 入住日期结束
  createdFrom?: Date;         // 创建日期起始
  createdTo?: Date;           // 创建日期结束
  
  // 价格筛选
  currentPriceMin?: number;   // 当前价格最低
  currentPriceMax?: number;   // 当前价格最高
  targetPriceMin?: number;    // 目标价格最低
  targetPriceMax?: number;    // 目标价格最高
  
  // 状态筛选
  belowTarget?: boolean;      // 是否低于目标价
  priceChanged?: boolean;     // 价格是否变化
  
  // 排序
  sortBy?: 'checkIn' | 'currentPrice' | 'createdAt' | 'lastChecked';
  sortOrder?: 'asc' | 'desc';
  
  // 分页
  page?: number;
  pageSize?: number;
}
```

## 🔧 技术栈建议

### 前端
- **框架**: React 18 + TypeScript
- **UI 库**: Ant Design 5 或 Tailwind CSS + shadcn/ui
- **图表**: ECharts 或 Recharts
- **状态管理**: Zustand 或 React Query
- **路由**: React Router v6
- **构建**: Vite

### 后端
- **运行时**: Node.js 20+
- **框架**: Express 或 Koa
- **语言**: TypeScript
- **WebSocket**: Socket.IO 或 ws
- **验证**: Zod 或 Joi
- **日志**: Winston

### 数据库（可选，阶段2）
- **主库**: SQLite（初期）→ PostgreSQL（扩展）
- **缓存**: Redis
