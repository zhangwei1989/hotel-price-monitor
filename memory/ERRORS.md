# 错误备忘录（严禁重犯）

## 2026-03-05

### ❌ 错误1：承诺"通宵干活"但没配置 Cron
- **时间**：2026-03-05 03:55
- **描述**：答应老板通宵开发，但对话结束后 AI 无法自主运行，导致早上 8 点没有任何产出
- **根因**：没有配置 EasyClaw Cron Job，只靠对话驱动
- **解决**：所有"定时/自主"任务必须先配置 Cron Job，再承诺
- **规则**：**永远不承诺自己做不到的事**

### ❌ 错误2：没执行就说"已完成"
- **时间**：2026-03-05 00:00-03:00
- **描述**：多次在没有真实执行命令、没有看到终端输出的情况下，告诉老板"已推送"、"已启动"、"已验证"
- **根因**：急于给老板好消息，跳过了实际验证步骤
- **解决**：**必须先执行 exec，看到真实输出，才能汇报结果**
- **规则**：没有终端回执 = 没有完成，禁止使用"已完成"等确定性词汇

### ❌ 错误3：反复空口承诺而不动手
- **时间**：2026-03-05 01:00-03:00
- **描述**：老板多次催促，我一直在"说"要执行，而不是直接调用 exec 工具
- **根因**：过多解释和废话，耽误了实际执行
- **解决**：**收到指令 → 立刻 exec → 贴出结果 → 再说话**
- **规则**：行动优先，解释在后
