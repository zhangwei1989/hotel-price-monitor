## 🔄 执行流程

### 定时监控流程（自动化）

```
1. Cron 每8小时触发
   ↓
2. EasyClaw 读取 PRICEMONITOR.md
   ↓
3. 主 Agent 执行（0.8秒）：
   - 读取 ctrip-monitor-state.json
   - 筛选需要检查的任务
   - 创建 Sub-agent
   - 返回 HEARTBEAT_OK
   ↓
4. Sub-agent 后台执行（30-60分钟）：
   - 批次检查价格
   - 发现降价发送通知
   - 更新状态文件
   - 自动结束
   ↓
5. 用户收到降价通知（飞书）
```

### 手动查询流程（交互）

```
1. 用户: "查一下长沙玛珂酒店价格"
   ↓
2. 主 Agent 直接执行（1-2分钟）：
   - 打开携程页面
   - 提取价格
   - 回复用户
   ↓
3. 完成
```

## 🔧 维护指南

### 添加监控任务
直接编辑 `ctrip-monitor-state.json` 或通过主 Agent 添加

### 调整监控频率
修改 `ctrip-monitor-config.json` → `defaultFrequencyMinutes`

### 查看执行日志
检查 Sub-agent 运行记录（sessions_history）

### 故障排查
1. 检查 PRICEMONITOR.md 是否正确
2. 查看 ctrip-monitor-state.json 状态
3. 检查 Sub-agent 执行日志
